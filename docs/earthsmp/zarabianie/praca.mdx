# Praca ⛏️

**Dostępne Prace:**

- Górnik
- Kopacz
- Drwal
- Rolnik
- Myśliwy
- Rybak
- Budowniczy

### Jak wziąć Prace?

Aby zacząć pracować wystarczy wejść w GUI pod `/praca` lub podejść do NPC na `/spawn` i wybrać Pracę w której chce się pracować!

### Jak opuścić Prace?

Aby opuścić pracę wystarczy że wpiszesz `/jobs leave <praca>`.

Gdy chcesz opuścić wszystkie prace wystarczy że wpiszesz `/jobs leaveall`.

:::info
Maksymalnie możesz mieć 3 prace na raz.
:::
